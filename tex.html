10 razones para usar TeX

1. Salida

Al finalizar de escribir tus textos obtendrás la más alta calidad tipografica (.dvi, .ps, .pdf) que cualquier procesador de documentos (LibreOffice Writer o Microsoft Word).

2. Composición tipográfica

TeX emplea sofisticados algoritmos tipográficos para construir párrafos y dividir palabras.

3. Rapidez

A diferencia de los procesadores de documentos que son lentos, pesados y con muchos errores, TeX es rápido. Requiere poca memoria RAM y espacio físico en disco duro.

4. Estabilidad

El creador de TeX —Donald E. Knuth— ha congelado el kernel del programa para garantizar que esté libre de errores.

5. Extensible

Aunque Donald E. Knuth decidió no hacer más cambios en TeX (1978), eso no significa que el programa no pueda ser adaptado a las nuevas necesidades tecnologías de hoy y mañana. El más claro ejemplo de ello es LaTeX, que extendió el TeX original de Knuth agregando nuevas características como referencias cruzadas, seccionado, indizado, generación automática de tabla de contenidos, cuadros, figuras, capítulos, secciones, subsecciones, subsubsecciones, párrafos, etcétera. Al igual que TeX, LaTeX también puede ser extensible a nuevas y modernas tecnologías.

6. La entrada es texto plano

Los archivos fuente que genera TeX (.tex) pueden ser portados a cualquier plataforma (Windows, Macintosh y sistemas *NIX). Al ser texto plano, cualquiera verlo y editarlo con cualquier editor de texto. Con archivos binarios (.doc, .docx, .odf), existe el problema de la compatibilidad y la retrocompatibilidad. Un archivo .docx puede no ser visualizado correctamente o editado con una versión anterior de Microsoft Word. Con TeX esto no sucede.

7. Salida multiple

Los archivos de salida que genera TeX son multiples: DeVice Independent (.dvi), PostScript (.ps), Portable Document File (.pdf), Hyper Text Markup Language (.html) y cualquiera que salga en el futuro. De eso se encargarán los desarrolladores sin que se afecte el formato, porque siempre será el mismo. Han pasado 46 años desde que se desarrolló TeX y todavía puedes ver, editar y compilar esos mismos archivos. Con la salida de una nueva version de Microsoft Word o LibreOffice Writer la compatibilidad y retrocompatibilidad se pierde.

8. Libre como la libertad

TeX es software libre como GNU, Linux, Apache, Perl, etcétera.

9. Multiplataforma

TeX puede ser ejecutado en cualquier sistema operativo: Windows, Macintosh y sistemas *UNIX.

10. Estandarización

No importa qué editor de texto plano (VI, VIM, Nano, etcétera) o plataforma uses (Windows, Macintosh, *UNIX, etcétera) para poder ver, editar y compilar los archivos.
